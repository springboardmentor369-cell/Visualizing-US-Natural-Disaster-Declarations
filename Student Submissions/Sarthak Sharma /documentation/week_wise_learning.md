### Overview

In the first week, the focus was on understanding how data is extracted, cleaned, and prepared for analysis. We were introduced to Power BI and learned why data quality is critical before performing any analysis or visualization.

Key Learnings

Learned how to extract data from different sources

Introduction to the Power BI interface and workflow

Understood that Power BI is a case-sensitive tool, which makes data standardisation very important

Data Cleaning Concepts (Power BI)

Text formatting techniques

Removing extra spaces

Standardising inconsistent text values

Splitting columns using a delimiter

Handling missing (null) values:

For non-numerical values, replace nulls with “Unknown”

For numerical values, replacing nulls using the average/mean

Removing null values where required

Power BI Concepts

Understanding Table View

Introduction to relationships between tables

Importance of clean and structured data for accurate dashboards

Outcome

This week built a strong foundation in data preparation, which is the most important step in the data analytics lifecycle.

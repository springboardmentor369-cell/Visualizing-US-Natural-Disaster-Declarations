# Dataset Information

This project uses FEMA Open Data API to fetch US Natural Disaster Declarations data.

Data Source:
FEMA Open Data Portal

API Used:
https://www.fema.gov/api/open/v2/DisasterDeclarationsSummaries

Note:
The dataset is dynamically fetched using an API key. 
For security reasons, the API key is not included in this repository.

To run this project:
1. Generate your own API key (if required).
2. Store it in a .env file.
3. Add your API key as:
   API_KEY=your_api_key_here
